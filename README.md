# ¡Hola! 西语启航 — 零基础西班牙语学习（v4）

纯本地网页版的零基础西班牙语学习软件，双击 `index.html` 即可使用（推荐 Chrome / Edge，口语识别需允许麦克风；所有口语/写作模式均支持文字输入作答）。

## 每日更新机制

- **每日解锁**：第 1 天开放 3 节语法课，之后每天自动 +2 节（成就页可开"自由模式"跳过）
- **每日任务清单**：首页每天生成 5~6 项任务（复习/词汇/语法/时态/口语，每 3 天一篇作文），全部完成额外 +15 XP
- **今日推荐**：首页自动指路——下一节该学的课、今天的写作题、记忆库待复习数量
- **每日一词**：每天轮换

## 🦙 宠物养成：西语小羊驼

- **赚金币**：每获得 1 XP 同步获得 1 🪙 金币；每攒 30 XP 随机掉落零食；完成每日全部任务额外送 🍖
- **商店**：4 种零食（饼干/牛奶/肉肉/小鱼干）+ 4 种玩具（毛线球/小足球/溜溜球/小草帽）
- **喂养互动**：喂食涨饱食度、陪玩涨心情、每天可摸头 3 次；饱食和心情每天自然下降，记得回来照顾它
- **六阶段成长**：🥚 神秘的蛋 → 🐣 破壳日 → 🦙 羊驼宝宝 → 少年羊驼 → 成年羊驼 → 👑 西语大师羊驼（喂食陪玩积累成长值）
- 羊驼会说西语：喂它会用西语道谢（¡Ñam ñam! ¡Delicioso!），可以改名（默认 Paco）

## 技术说明

- 纯 HTML + CSS + JS，无依赖：`index.html` + `js/data.js`（基础）+ `js/data2.js`（进阶）+ `js/data3.js`（现西一册拓展）+ `js/data4.js`（现西二三册 + 宠物配置）+ `js/app.js`（逻辑）+ `css/style.css`
- 口语例句取材《现代西班牙语》经典课文句型（¿Quién es él? / ¿Cuántas personas hay en tu familia? / 作息与钟点表达等）
- 朗读：浏览器语音合成（es-ES）；口语打分：Web Speech API 识别 + 编辑距离相似度
- 作文批改：字数 + 句型正则（去重音归一化后匹配）
